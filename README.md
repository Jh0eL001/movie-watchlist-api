# Movie Watchlist API  

My first API to manage a movie watchlist, built following best practices in Back-end Development
API REST robusta para gestionar una lista de películas personal, construida siguiendo las mejores prácticas de desarrollo Backend.

## Tech

* **Node.js** & **Express** - Server Framework (Framework de servidor).
* **Prisma ORM** - DB Management (Gestión de base de datos PostgreSQL).
* **JWT (JSON Web Tokens)** - Secure Auth(Autenticación segura).
* **Zod** - Data Validation (Validación de datos).
* **PostgreSQL** - Relational Data Base (Base de datos relacional).
